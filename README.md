#  Game Data Generator 

这是一个基于 C# 开发的数据自动生成与验证工具，主要用于批量生成游戏道具配置数据，并导出为 Excel 表格进行逻辑验证。

## 项目简介
在前期游戏开发或测试阶段，经常需要大量模拟数据来验证配置表的读取逻辑。本项目通过 C# 代码自动生成包含 ID、名称、等级、属性等字段的随机道具数据，并直接导出为标准的 Excel 配置表，实现了“代码造数据 -> 导出验证”的完整闭环。

## 核心功能
- ✅ **批量数据生成**：可根据设定规则，一键生成数百条不重复的游戏道具模拟数据。
- ✅ **Excel 导出**：使用 EPPlus 库将内存中的数据对象直接写入 Excel 文件，格式规范。
- ✅ **数据验证**：通过逆向生成配置表，快速验证数据生成逻辑的健壮性与正确性。
- ✅ **轻量高效**：纯 C# 控制台应用，无多余依赖，运行速度快。

## 技术栈
- C# / .NET
- EPPlus (用于 Excel 文件的读写操作)

## 如何运行
1. 确保本地已安装 .NET 运行环境。
2. 克隆或下载本项目到本地。
3. 使用 Visual Studio 打开 `.sln` 解决方案文件。
4. 编译并运行项目，程序会自动在指定路径生成包含道具数据的 Excel 表格。
